# 🌤️ Weather App
A simple and responsive weather application built using HTML, CSS, and JavaScript. Users can search for any city and get real-time weather information including temperature, weather conditions, and humidity.
## 🚀 Live Demo
https://weather-olgjric11-snehahiremath-859s-projects.vercel.app
## 📂 GitHub Repository
https://github.com/SnehaHiremath-859/weather-app
## ✨ Features
- Search weather by city name
- Displays current temperature
- Shows weather conditions
- Displays humidity information
- Responsive and mobile-friendly design
- Clean and user-friendly interface
## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
- OpenWeather API
## 📦 Installation
1. Clone the repository:
```bash
git clone https://github.com/SnehaHiremath-859/weather-app.git
```
2. Open the project folder.
3. Open `index.html` in your browser.
